About Order and Chaos

Order and Chaos is a board game played by two players. One players goal is to build columns of 5. The other players goal is to prevent this from happening. Each player can play both colors.

Instructions

To play a color hover over the field you want to set and the colors to click on will appear. The red is in the left half of the field and the blue is in the right half of the field. Once a field is set this way it can only be removed with the undo function

To start a new game you can click the "New" button in the navigation bar and the board will be reset. You can still return to your previous game state with the "Undo" button.

Once a row of five in horizontal, diagonal or vertical direction exists Order wins. It is not possible to set any fields after this. The only options are to undo the last turn or start a new game.

If it is impossible for Order to win Chaos automatically wins. The game will also block any fields from being set should this be the case.
